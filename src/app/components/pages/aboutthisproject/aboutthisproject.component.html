Sobre esta página

Está pagina foi desenvolvida para apresentação da minha carreira e dos meus conhecimentos, mas também como forma de praticar os estudos de Angular e TypeScript.
Ela envolve o desenvolvimento com Angular atualizada na versão 17, versionada em um repostório no GitHub, o deploy e publicação acontece de forma automática usando o GitHub Actions sendo hospedada no Azure Static Web Apps.
